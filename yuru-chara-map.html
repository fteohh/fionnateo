Yuru-chara Map

Student Project | Spring 2021


Yuru-chara map is a web app built with HTML, CSS, Javascript, Heroku, and Mapbox that allows users to explore Japan through its unique mascots, or yuru-chara.
Feel free to view and interact with the project here!


The Problem

Inspired by my summer studying abroad in Japan, Yuru-chara Map is a web app I designed and built in a web development course that aims to facilitate exploratory learning through environmental interactions. There are currently no existing platforms that have data on all yuru-chara and are English accessible. Yuru-chara bridges this language barrier and offers a fun educational experience to travelers, students, and other curious people.

Reflection

I experienced many firsts in completing this project. It was my first time developing a web app, coding with Mapbox, and working on my own project from ideation to implementation. Some of the most important lessons I learned are:


Define the product

Going into the project, I had a strong idea of what value the web app would bring but possessed a vague design. I jumped right into development, but this slowed my progress because I was making design and development choices in parallel. However, I quickly realized the importance of dividing up the creation process and allocating time to fully focus on individual phases.


Budget your time

Working with a limited timeline and skills, I had to set clear goals for the project. After experiencing a mismatch between my expectations and execution, I tried determining my “delighters” and needs and working on a single slice of the project for each weekly checkpoint.
