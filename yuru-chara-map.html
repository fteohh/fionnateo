Yuru-chara Map
Student Project | Spring 2021


Tools Used: HTML, CSS, JavaScript, Heroku, Mapbox
Feel free to view and interact with the project here!


Task: Investigate web development technologies and implement a project of your choosing.

When I was studying abroad in Japan, I saw mascots everywhere and ended up developing a personal interest in them. So, I decided to make an interactive map that allows users to “explore” Japan and learn more about various places through their mascots, since they usually incorporate motifs that represent local culture, history, or production.

Basic features: This web app allows users to interactively explore Japan and highlights the special qualities of each region and prefecture.


Takeaways:

Planning and designing a project involves so many components. I realized how helpful it is to sketch interfaces and break everything down into smaller components before implementing them. It was also a massive amount of fun to explore new web technologies and implement a project I'm passionate about. I'm excited to create more projects in the future!
